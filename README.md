## アプリ名：IdeaStack

## 概要
アイデアを投稿・整理できる個人開発者向けのWebアプリです。  
ログイン後、自分のアイデアを「メモ・ToDo・ストック」などに分類し管理できます。

--

## 使用技術スタック
- フロントエンド：HTML / CSS / Bootstrap
- バックエンド：Python（Flask）
- データベース：MySQL

--

## 主な機能
- ユーザー登録・ログイン機能
- アイデアのCRUD（投稿・編集・削除）
- ステータス分類（未開発 / 開発中 / 完了）
- ステータス絞り込み & キーワード検索

--
