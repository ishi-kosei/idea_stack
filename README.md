## IdeaStackアプリ

アイデアを投稿・整理できる個人開発者向けのWebアプリです。  
ログイン後、自分のアイデアを「メモ・ToDo・ストック」などに分類し管理できます。

---

## 主な機能
- ✅ユーザー登録・ログイン機能
- ✅アイデアのCRUD（投稿・編集・削除）
- ✅ステータス分類（未開発 / 開発中 / 完了）
- ✅ステータス絞り込み & キーワード検索

---

## 使用技術

- Python 3.11
- Flask
- Flask-SQLAlchemy
- Flask-Login
- Flask-WTF
- MySQL
- Bootstrap 5
- Docker / Docker Compose
